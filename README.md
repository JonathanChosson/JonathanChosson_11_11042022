## Installation

## Comment lancer l'application en local ?

### Lancer le frontend :

Allez au repo cloné :

```
$ cd JonathanChosson_11_11042022
```

Installez les packages npm (décrits dans `package.json`) :

```
$ npm install
```

Lancer un serveur local :

```
$ yarn start
```
